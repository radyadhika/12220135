## Menjalankan aplikasi secara lokal
### Install streamlit via pip
```
pip install streamlit
```
### Install streamlit via conda
```
conda install -c conda-forge streamlit
```
### Run
```
streamlit run UAS.py
```
## Required packages
- Streamlit
- Pandas
- Numpy
- Matplotlib
